# 🛒 E-Commerce Data Cleaning Proyect

## Project Overview
Data cleaning and organization of 103 e-commerce records to
identify and correct data quality issues.

## Poblems Found & Fixed
- 23 total errors identified
- 21 errors corrected
- 2 records flagged as insufficient data

## Error Types Corrected
- Inconsistent category names (ELECTRONICS - Electronics)
- Text values in numeric fields (abc, four hundred, 300$)
- Negative quantities and prices
- Invalid dates
- Empty cells
